# PDFCraft — Free PDF Tools

13 free PDF tools that run entirely in your browser. No sign up. No uploads. No cost.

## Tools
- Merge PDF
- Split PDF
- Compress PDF
- PDF to Image
- Rotate PDF
- Watermark PDF
- Protect PDF
- Unlock PDF
- JPG to PDF
- Delete Pages
- Add Page Numbers
- Reorder Pages
- PDF to Grayscale

## Tech Stack
- Next.js 16
- pdf-lib
- pdfjs-dist
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy
Hosted on Cloudflare Pages.
